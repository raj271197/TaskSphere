# 🚀 Quick Start Guide

Get the project running in 5 minutes!

## Step 1: Clone the Repository

```bash
git clone https://github.com/raj271197/finalexam.git
cd finalexam
```

## Step 2: Install Dependencies

```bash
npm install
```

## Step 3: Run Development Server

```bash
npm run dev
```

## Step 4: Open in Browser

Navigate to **http://localhost:3000**

That's it! 🎉

## 🎨 Features to Try

### Counter
- Click **➕ Increment** to increase the count
- Click **➖ Decrement** to decrease the count
- Click **🔄 Reset** to reset to zero
- Watch the dynamic feedback messages!

### Todo List
- Type a task in the input field
- Press **Enter** or click **Add Task**
- Click **Delete** to remove a task
- Watch smooth animations!

### Posts
- Posts automatically load from the API
- Hover over posts to see elevation effect
- Each post shows ID, title, and body

### Navigation
- Hover over nav links to see animated underline
- Click to navigate between pages
- Navbar stays at the top (sticky)

## 🐛 Troubleshooting

### Port Already in Use
If port 3000 is already in use:
```bash
npm run dev -- -p 3001
```

### Dependencies Issues
Clear cache and reinstall:
```bash
rm -r node_modules package-lock.json
npm install
```

### Build Errors
Make sure you have Node.js 18+:
```bash
node --version
```
